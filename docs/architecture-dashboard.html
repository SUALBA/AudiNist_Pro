Vision

AuditEngine is a lightweight cybersecurity audit engine designed to transform complex frameworks into practical evaluation workflows. Instead of navigating large framework documentation, auditors can review structured controls and record their assessment in a consistent way.

Execution Flow


User starts audit
       ↓
Framework adapter loads controls
       ↓
AuditEngine creates audit session
       ↓
User evaluates controls
       ↓
EvaluationManager records results
       ↓
RiskEngine calculates risk
       ↓
ReportEngine generates report

Engine API


const engine = new AuditEngine(nistAdapter)

engine.startAudit({
 company: "Example Corp",
 auditor: "Susana",
 framework: "NIST"
})

engine.evaluateControl("PR.AC-1", {
 status: "Partial",
 notes: "Identity management incomplete"
})

engine.generateReport()

Product Evolution Roadmap

  • improved reporting
